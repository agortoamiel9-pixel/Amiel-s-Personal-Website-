# Amiel's Personal Website

Welcome to my personal website! This is a simple, clean portfolio site to showcase my work and information about me.

## Features

- Simple and responsive design
- Easy to customize
- Clean HTML and CSS structure

## Files

- `index.html` - Main page with About Me, Projects, and Contact sections
- `style.css` - Styling for the website
- `README.md` - This file

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information
4. Deploy to GitHub Pages or any web hosting service

## Customization

Feel free to:
- Update the content in `index.html`
- Modify the colors and styling in `style.css`
- Add new sections as needed
